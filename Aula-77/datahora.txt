Resumo da Aula: Manipulação de Datas
A Classe Date: No JavaScript, trabalhamos com datas instanciando um objeto da classe Date. Ao fazer new Date(), o JS captura a data e hora exatas do sistema (incluindo o fuso horário).

Métodos "Get": São funções usadas para extrair informações específicas, como:

getDate(): Dia do mês (1-31).

getMonth(): Mês (0-11, onde 0 é Janeiro).

getFullYear(): Ano com 4 dígitos.

getDay(): Dia da semana (0-6, onde 0 é Domingo).

Timestamp (Unix Time): O professor explica que o método Date.now() retorna o número de milissegundos passados desde 1 de janeiro de 1970. Esse valor é incremental e nunca se repete, sendo útil para cálculos de tempo.

Formatação Manual: Como os métodos get retornam números puros, o professor ensina a tratar os dados (ex: adicionar o "0" à esquerda para dias menores que 10) usando operadores ternários para criar uma string formatada (dd/mm/aaaa).



Explicação do Código Linha por Linha
Abaixo, explico o trecho onde o professor cria uma data formatada e a exibe no HTML:

// 1. Criamos um novo objeto de data com as informações atuais do sistema
const data = new Date();

// 2. Extraímos o dia do mês e tratamos se for menor que 10 (ex: transforma 3 em "03")
let dia = data.getDate();
dia = dia < 10 ? "0" + dia : dia; 

// 3. Extraímos o mês. IMPORTANTE: O JS conta meses de 0 a 11, por isso somamos +1
let mes = data.getMonth() + 1;
mes = mes < 10 ? "0" + mes : mes;

// 4. Extraímos o ano com 4 dígitos (ex: 2024)
const ano = data.getFullYear();

// 5. Montamos uma string final unindo as variáveis com a barra "/"
const data_formatada = dia + "/" + mes + "/" + ano;

// 6. Selecionamos um elemento no HTML e injetamos a data formatada
document.getElementById("div_data").innerHTML = data_formatada;


Principais Métodos Citados:
data.getTime(): [09:11] Retorna o timestamp (milissegundos).

data.toDateString(): [09:53] Retorna apenas a parte da data em formato americano (ex: "Sat Dec 03 2022").

data.toLocaleDateString("pt-BR"): (Dica extra não aprofundada na aula, mas útil) Formata automaticamente para o padrão brasileiro.

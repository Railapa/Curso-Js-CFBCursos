Nesta aula 115 do CFBCursos, o professor Bruno continua a exploração sobre Desestruturação (Destructuring) em JavaScript, focando em casos mais avançados e práticos.

Aqui estão os pontos principais abordados nesta segunda parte:

1. Ignorando Elementos em Arrays
Você não precisa extrair todos os itens de um array. Ao deixar espaços vazios entre as vírgulas, o JavaScript ignora os valores nessas posições:

Sintaxe: let [a, , b] = [10, 20, 30];

Resultado: a será 10 e b será 30. O valor 20 foi ignorado.

2. Uso do Operador Rest (...)
O operador rest é extremamente útil quando você quer capturar os primeiros elementos e agrupar o restante em um novo array:

Sintaxe: let [a, b, ...resto] = [10, 20, 30, 40, 50];

Resultado: a é 10, b é 20, e resto será o array [30, 40, 50].

3. Desestruturação de Objetos com Novos Nomes de Variáveis
Vimos que, por padrão, as variáveis devem ter o mesmo nome das chaves do objeto. Porém, você pode renomeá-las durante a extração:

Sintaxe: const {cidade: local, preco: valor} = meuObjeto;

Resultado: O valor da chave cidade vai para a variável local.

4. Extração de Objetos Aninhados
Se um objeto estiver dentro de outro, você pode navegar pela estrutura para pegar o dado que deseja:

Exemplo: Se você tem um objeto usuario com um objeto endereco dentro, pode extrair apenas a rua assim: const { endereco: { rua } } = usuario;.
